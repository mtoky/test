About This Project

This is a sample Python web application built with Flask to demonstrate deployment on Scalingo's hosting platform.

Scalingo provides a Platform as a Service (PaaS) that makes it easy to deploy web applications in various programming languages, including Python.

Key features of Scalingo's Python hosting:

  • Support for different Python versions
  • Automatic detection of Python projects
  • Support for various dependency management tools (Pipenv, requirements.txt, Poetry)
  • Easy scaling of your application
  • Built-in database services
